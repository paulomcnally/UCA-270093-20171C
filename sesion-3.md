**Instalar LoopbackJS - CLI**

    $ npm install -g loopback-cli

**Crear un API**

    $ lb

**Crear un modelo**

    $ lb model

**Crear una relación entre modelos**

    $ lb relation

**Correr/ejecutar el API**

    $ npm start
